1. Download the widget jar file
  2. Inherit LightboxImage widget module.
    In your GWT module xml (For ex. KitchenSink.gwt.xml) under the line
    <inherits name='com.google.gwt.user.User'/> add line
    <inherits name='therandomhomepage.widgets.LightboxImage'/>
  3. Please make sure that widget jar file is available in your GWT module classpath.

Note: No need to include javascript files in your GWT module
For ex: <script type="text/javascript" src="js/prototype.js"></script>
There are some modifications to original Lightbox JS to work in GWT and all the required javascript libraries (prototype.js, effects.js & swfobject.js) & stylesheets are injected into the LightboxImage Widget module for easier plugging. Enjoy...
