Interface HasHTML

public interface HasHTML
extends HasText

// Implemented by Hyperlink, HTML, DialogBox, MenuItem, TreeItem, ButtonBase
An object that implements this interface contains text, which can be set and retrieved using these methods. The object's text can be set either as HTML or as text.

Methods

getHTML()Gets this object's contents as HTML.
setHTML(String)Sets this object's contents via HTML.

Method Detail

Return Value

setHTML

public void setHTML(String html)
Sets this object's contents via HTML. Use care when setting an object's HTML; it is an easy way to expose script-based security problems. Consider using setText whenever possible.

Parameters
