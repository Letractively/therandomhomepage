Command-line Tools

GWT comes with a few handy command-line tools to get you up and running quickly. They are also useful for adding new things to existing projects. For example, projectCreator could be used to make an Eclipse project for one of the samples that comes with GWT.
  • projectCreator
    Generates a basic project skeleton and an optional Ant buildfile and Eclipse project.
  • applicationCreator
    Generate a starter application.
  • junitCreator
    Generate a JUnit test.
  • i18nCreator
    Generate an i18n properties file and synchronization script.
