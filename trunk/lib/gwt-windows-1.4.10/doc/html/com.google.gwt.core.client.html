Package com.google.gwt.core.client

Fundamental classes used in client-side GWT code. The classes in this package represent concepts fundamental to GWT, such as module entry points, and interfacing with JavaScript.

Classes

GWTSupports core functionality that in some cases requires direct support from the compiler and runtime systems such as runtime type information and deferred binding.
JavaScriptExceptionAny JavaScript exceptions occurring within JSNI methods are wrapped as this class when caught in Java code.
JavaScriptObjectAn opaque handle to a native JavaScript object.

Interfaces

EntryPointImplement this interface to allow a class to act as a module entry point.
