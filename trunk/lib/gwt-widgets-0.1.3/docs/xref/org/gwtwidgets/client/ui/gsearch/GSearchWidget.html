30  /*
31   * Provides a Widget surrounding the Google Ajax Web Search component.
32   * Code structure inspired by GMap2 implementation by algaforge
33   * http://sourceforge.net/projects/gwt/
34   * Preference is for lazy implementation of objects.
35   * 
36   * Functionality reflects that from the Google AJAX Search API
37   * (http://code.google.com/apis/ajaxsearch/documentation/reference.html)
38   * 
39   * Implementation from user side follows the requirement of expected order 
40   * of operations as:
41   * 
42   * 		sc = new GSearchControl();
43   * 		sc.addSearcher();
44   * 		sc.draw();
45   * 
46   * Once these steps have occurred, the search control is active and ready to begin performing searches. 
47   *
